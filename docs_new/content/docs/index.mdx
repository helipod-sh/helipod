stackbase is an open-source backend you host yourself. You write TypeScript **query**,
**mutation**, and **action** functions, and stackbase runs them server-side, transactionally, and
**pushes updated results to every subscribed client in real time**. No polling, no
cache-invalidation code to write yourself.

It's TypeScript end to end: engine, CLI, and client SDK. It runs on SQLite or Postgres, and deploys
anywhere you can run a container or a single binary. No vendor lock-in.

## What you get

- **A reactive database.** Subscribe to a query and it stays up to date automatically.
- **Server functions.** Queries, mutations, and actions, with typed arguments and return values.
- **A client SDK.** `useQuery`/`useMutation` hooks, optimistic updates, and durable offline sync.
- **Opt-in components.** Authentication, authorization, notifications, scheduling, durable
  workflows, and change triggers, composed into your project as you need them.
- **A CLI and dashboard.** Hot-reloading local development, a live data browser, and one-command
  self-hosting.
